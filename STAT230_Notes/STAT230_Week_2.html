Mutually Exclusive Events

2 events are mutually exclusive if only one of them can occur at a given time. In other words, there are no common elements in the 2 sets.

Probability Set Function

Suppose the function P associates a real value, P(A), with each event A defined on a sample space S such that:

  1. 0 ≤ P(A) for every element A
  1. P(S) = 1
  1. If A1, A2, …, An are mutually exclusive events, P(A1 U A2 U…U An) = P(A1) + P(A2) + …. + P(An)

then, P is called the probability set function and P(A) is called the probability of A. The 3 rules are called axioms of probability and hold in every scenario, no matter what definition of probability we use.

Properties of Probabilities

Let S be the Sample Space for an experiment. Let A, B, A1, A2 be defined on S. Then:

  1. 0 ≤ P(A) ≤ 1
  1. P(A) + P(~A) = 1 [~A is the compliment of A]
  1. If A is a subset of B, then P(A) ≤ P(B)
  1. If A and B are 2 mutually exclusive events, then P(A U B) = P(A) + P(B) [can be extended for multiple events]
  1. P(A U B) = P(A) + P(B) - P(A intersection B) [can be extended for multiple events, you’ll have to draw a Venn diagram and figure things out]

Independant Events

We call 2 events A and B independant if and only if P(A and B) = P(A)P(B). All independant events have the following properties.

  1. P(A1 and A2 and … and An) = P(A1)P(A2)…P(An)
  1. A and B are independant events if and only A and ~B are independant events. Similarly A and ~B or ~A and ~B
